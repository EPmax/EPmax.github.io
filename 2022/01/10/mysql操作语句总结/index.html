常用MySql操作语句总结

2022年01月10日 956 字 大概 4 分钟

常用MySql操作语句总结

  • 一、DDL操作数据库、表

    1、操作数据库：

    进入数据库：mysql -uroot -p/ mysql -u 用户名 -p 密码 -h 要连接的mysql服务器的ip地址 -p 端口号

    退出：exit\quit

    卸载：mysql -remove mysql

    显示当前数据库下的所有数据库名称：show databases

    在当前数据库下创建子数据库：create database if not exists 数据库名称

    删除数据库：drop database if exists 数据库名称

    进入数据库：use 数据库名称

    查看当前使用的数据库：select database()

    2、操作表：

    显示当前数据库下所有的表：show tables

    查询表的结构：desc 表名称

    创建表：create tables 表名(

    字段名1 数据类型,

    字段名2 数据类型,

    字段名3 数据类型,

    ……,

    字段名n 数据类型

    );\\注意最后一行不加逗号

    删除表：drop table if exists 表名

    修改表名：alter table 表名 rename to 新的表名

    添加一列：alter table 表名 add 列名 数据类型

    修改数据类型：alter table 表名 modify 列名 新的数据类型

    修改列名和数据类型：alter table 表名 change 新列名 新的数据类型

    删除表列：alter table 表名 drop 列名

  • 二、DML操作数据（增、删、改）

给指定列添加数据：instert into 表名(列名1,列名2,列名3,…….) values(值1,值2,……);

修改表数据：updata 表名 set 列名1=值1,列名2=值2,…… where 条件

删除表数据：delete from 表名 where 条件\\不加where条件将删除所有数据

  • 三、DQL查询操作

    • 1、基础查询

      查询所有：select * from 表名;

      按列查询：select 列名1,列名2,…… from 表名;

      去除重复记录：select distinct 列名1,列名2,…… from 表名;

      起别名：as(as可以省略)

    • 2、条件查询

      按列查询：select 列名1,列名2,…… from 表名 where 条件1 and 条件2 and……

      • between语句 等价于 …>=x>=…；可用于按日期条件查询

      • in语句的使用：select 列名1,列名2,…… from 表名 where 字段名 in(1,2,3,
        …)等价于 select 列名1,列名2,…… from 表名 where 字段名=1 or 字段名=2 or 字段名=3 or …

      • null数据的比较不能使用=和!=，要使用is和is not。

      • 符号 功能
        > 大于
        < 小于
        >= 大于等于
        <= 小于等于
        = 等于
        <>或!= 不等于
        between…and… 在某个范围之内，包含边界
        in() 多选一
        like 占位符 模糊查询，_代表单个任意字符，%代表多个任意字符
        is null 是null
        is not null 不是null
        and和&& 并且
        or和|| 或者
        not和! 非，不是

  • 3、分组查询

    • 1、聚合函数

      • 概念：将一列数据作为一个整体，进行纵向计算

      • 语法：select 聚合函数名(列名) from 表名;

      • 聚合函数：

        函数名 功能
        count(列名) 统计数量
        max(列名) 最大值
        min(列名) 最小值
        sun(列名) 求和
        avg(列名) 平均值

      • 注意：null不参与所有聚合函数的运算

    • 2、分组查询

      • 语法： select 聚合函数名(列名) from 表名 where条件（可省略）group by 分组字段名 having条件(分组后条件过滤，可省略) ; 例：select sex,avg(math) from 表名 group by sex;

  • 4、排序查询

    排序查询语句：按列查询：select 列名1,列名2,…… from 表名 order by 排序字段名1 排序方式（可省略）,排序字段名2 排序方式（可省略）,…

    • 排序方式：

      • asc(升序)

      • desc(降序)

    • 如果排序字段值一样，则按下一个排序字段的值进行排序

  • 5、分页查询

    • 语法：select 列名1,列名2,…… from 表名 limit 起始索引,查询条目;

    • 起始索引=(当前页码-1)*每页显示的条数

    • 注意：limit是MySQL的方言，Oracle使用rownumber，SQL Server使用top
